# SearchService

Handles search index updating

## /addVideo

_Videos can only be created on channels owned by the author_

Adds a video to the Algolia search index

## /updateVideo

_Videos can only be updated by the author or the channel owner_

Updates a video in the Algolia search index

## /removeVideo

_Videos can only be removed by the author or the channel owner_

Removes a video from the Algolia search index

## /addChannel

Adds a channel to the Algolia search index

## /updateChannel

_Channels can only be updated by the owner_

Updates a channel in the Algolia search index

## /removeChannel

_Channels can only be removed by the owner_

Removes a channel from the Algolia search index
